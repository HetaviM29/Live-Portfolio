## AI-Resistant Phishing and Deception Detection System

### Problem Solved

Modern phishing attacks use increasingly sophisticated techniques that are difficult to detect using traditional rule-based methods. The goal of this project was to develop a machine learning-based system capable of detecting phishing or deceptive content automatically.

---

### Description

This project focuses on building an intelligent phishing detection system that analyzes textual and structural features of suspicious content.

The system preprocesses input text, extracts relevant features, and applies classification algorithms to determine whether content is legitimate or malicious.

The project explores how machine learning can improve cybersecurity systems and automate detection processes.

---

### Tech Stack

Programming:
- Python

Machine Learning:
- Supervised Learning
- Text Classification
- Feature Engineering

Libraries:
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

Techniques:

- Text preprocessing
- Tokenization
- Feature extraction
- TF-IDF vectorization
- Classification models

Models:

- Logistic Regression
- Naive Bayes

Tools:

- Jupyter Notebook

---

### Challenges

- Selecting meaningful features for phishing detection
- Dataset imbalance
- Improving classification accuracy
- Avoiding false positives

---

### Outcomes

- Built a functional phishing detection model
- Implemented feature-based classification
- Gained experience in cybersecurity applications of AI

## AVL Dictionary

### Problem Solved

Standard dictionary implementations using simple binary search trees can become unbalanced, leading to inefficient search operations. A balanced structure is required to maintain consistent performance.

---

### Description

Developed a reusable and modular C++ library implementing a dictionary using AVL trees to ensure balanced and efficient search operations.

The system supports insertion, deletion, keyword updates, and sorted traversal of dictionary entries.

---

### Tech Stack

Programming:

- C++

Concepts:

- AVL Trees
- Self-balancing Binary Search Trees
- Dictionary Data Structure
- Tree Traversal

---

### Challenges

- Maintaining AVL tree balance after insertion and deletion
- Implementing correct tree rotations

---

### Outcomes

- Built a reusable AVL-based dictionary library
- Improved understanding of balanced tree data structures
